## 1. 产品定位和目标用户

### 1.1 产品定位

Notebook 是一款轻量级 Web 版 Markdown 笔记工具，专注于提供简洁、高效的本地笔记管理体验。产品核心理念是「专注写作，本地优先」，所有数据存储在用户本地文件系统，无需云端同步，保护用户隐私。

### 1.2 目标用户

| 用户类型 | 特征描述 | 核心需求 |
|---------|---------|---------|
| 开发者 | 习惯使用 Markdown，需要记录技术笔记、代码片段 | 快速新建、编辑、搜索笔记 |
| 学生/研究者 | 需要整理学习笔记、文献摘要 | 结构化存储、全文搜索 |
| 写作者 | 习惯 Markdown 写作，需要简洁的写作环境 | 无干扰编辑、本地保存 |
| 知识管理者 | 需要管理个人知识库 | 批量管理、快速检索 |

### 1.3 产品愿景

成为用户本地首选的 Markdown 笔记管理工具，在简洁性和功能性之间取得平衡，让用户专注于内容创作而非工具本身。

## 3. 用户操作流程

### 3.1 新建笔记流程

```
用户进入首页
    ↓
点击「新建笔记」按钮 (或按 Ctrl+N)
    ↓
系统自动生成文件名 (格式：YYYY-MM-DD-HH-mm-ss.md)
    ↓
打开空白编辑界面，光标定位到编辑区
    ↓
用户输入笔记内容
    ↓
系统自动保存 (每 30 秒) 或用户手动保存 (Ctrl+S)
    ↓
保存成功，列表更新显示新笔记
```

### 3.2 编辑笔记流程

```
用户在笔记列表中点击目标笔记
    ↓
系统加载笔记内容到编辑器
    ↓
用户编辑内容（左侧编辑区，右侧预览区）
    ↓
系统实时渲染预览
    ↓
用户完成编辑，点击保存或自动保存触发
    ↓
更新文件修改时间，返回列表
```

### 3.3 搜索笔记流程

```
用户在搜索框输入关键词
    ↓
系统实时搜索（或按 Enter 触发）
    ↓
搜索结果按相关性排序展示
    ↓
匹配的文件名和内容片段高亮显示
    ↓
用户点击搜索结果，打开对应笔记
```

### 3.4 删除笔记流程

```
用户在笔记列表中选择目标笔记
    ↓
点击删除按钮（或右键菜单选择删除）
    ↓
系统弹出二次确认对话框
    ↓
用户确认删除
    ↓
文件移动到回收站或直接删除（根据配置）
    ↓
列表刷新，笔记消失
```

### 4.2 组件说明

#### 4.2.1 顶部导航栏
- **Logo 区**：左侧显示产品 Logo 和名称「Notebook」
- **搜索框**：居中位置，支持实时搜索，placeholder 为「搜索笔记...」
- **功能按钮**：右侧显示设置按钮、主题切换按钮

#### 4.2.2 左侧边栏（笔记列表）
- **宽度**：固定 280px，可折叠
- **新建按钮**：顶部显眼的「+ 新建笔记」按钮
- **笔记项**：每行显示文件名、修改时间、内容预览（前 50 字符）
- **文件夹**：支持树形结构展开/折叠
- **选中状态**：当前编辑的笔记高亮显示

#### 4.2.3 主编辑区
- **布局**：左右分栏，编辑区和预览区各占 50%
- **编辑区**：
  - 支持 Markdown 语法高亮
  - 行号显示
  - 自动保存状态指示器
  - 字数统计
- **预览区**：
  - 实时渲染 Markdown 内容
  - 支持代码块语法高亮
  - 支持表格、列表、引用等格式

#### 4.2.4 底部状态栏
- **左侧**：笔记总数统计
- **中间**：最后保存时间
- **右侧**：手动保存按钮、自动保存开关

### 4.3 交互细节

| 交互场景 | 交互方式 | 反馈效果 |
|---------|---------|---------|
| 新建笔记 | 点击按钮或 Ctrl+N | 淡入动画，光标自动聚焦编辑区 |
| 保存笔记 | Ctrl+S 或自动触发 | 右下角 Toast 提示「已保存」，状态栏更新时间 |
| 搜索笔记 | 输入时实时搜索 | 搜索结果高亮，无结果时显示空状态 |
| 删除笔记 | 删除按钮 → 二次确认 | 确认对话框，确认后列表项淡出动画 |
| 切换主题 | 点击主题按钮 | 平滑过渡动画，主题偏好持久化 |
| 折叠边栏 | 点击折叠按钮 | 边栏滑出，编辑区自动扩展 |

### 4.4 响应式设计

| 屏幕宽度 | 布局方案 |
|---------|---------|
| ≥ 1200px | 三栏布局：边栏 + 编辑区 + 预览区 |
| 768px - 1199px | 两栏布局：边栏 + 编辑区（预览通过标签页切换） |
| < 768px | 单栏布局：编辑区全屏，预览通过按钮切换显示 |
