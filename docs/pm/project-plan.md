## 🚀 开发原则

**核心方针:** 不设定具体工时和完成时间，每个任务以最快速度完成

| 原则 | 说明 |
|------|------|
| ⚡ 速度优先 | 每个任务尽快完成，不等待 |
| 🔄 并行开发 | 前后端并行，能同时做的同时做 |
| 📦 小步快跑 | 完成一个功能立刻联调，不堆积 |
| ✅ 即时验收 | 每个 Phase 完成后立刻演示验收 |
| 🎯 结果导向 | 关注交付物，不关注过程时长 |
