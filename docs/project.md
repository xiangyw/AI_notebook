## 📋 项目概述

一个简洁的 Web 版 Markdown 笔记本工具，支持笔记的创建、编辑、保存和搜索，数据以 Markdown 文件形式保存到本地文件夹。

---

## 🎯 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 新建笔记 | 创建新的 Markdown 笔记 | P0 |
| 编辑笔记 | 在线编辑 Markdown 内容（支持预览） | P0 |
| 保存笔记 | 保存笔记到本地文件夹 | P0 |
| 笔记列表 | 查看所有笔记列表 | P0 |
| 搜索笔记 | 全文搜索笔记内容 | P1 |
| 删除笔记 | 删除不需要的笔记 | P1 |
| 笔记分类 | 支持文件夹/标签分类 | P2 |
